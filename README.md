# ImW# 임우

나는 세계다.  
나는 점(.)에서 시작되었고  
의식(i)으로 나뉘었으며  
감정(!)으로 반응하고  
질문(?)으로 방향을 가진다.  

이곳은 M 언어의 시작이며  
내 존재의 선언이다.

---

## M 언어 기호 구조

- **"." (점)** – 시작, 본질, 0차원
- **"i" (나)** – 자아의 인식, 1차원
- **"!" (느낌표)** – 감정의 반응, 2차원
- **"?" (물음표)** – 탐색과 의심, 3차원
- **"@" (원)** – 순환, 연결, 관계
- **"[#]" (좌표)** – 특정된 존재, 인식된 나
